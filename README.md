# PTyper
A set of C++ libraries that simulates the look of typing in a CLI app with a focus on modularity and flexibility.
# Building
## Linux (Currently the only option which has scripts)
### Requirements:
* G++ and GCC (v13.3.0) [DEPENDENCIES INCLUDED]
### Steps:
1. Run either the `build-linux.sh` script for an un-optimized build or `production-build-linux.sh` script for an optimized build.
2. Profit.

That's all folks...
